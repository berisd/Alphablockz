Alphablockz is a 2D word game.
It requires a Java Runtime v1.6 or later.

Visit the Homepage at http://www.beris.at
Please send your appreciated comments and feedback to bernd.riedl@gmail.com

For instructions click on the Help-button or press the key H in the mainmenu.

To Start the game:
For Windows execute run.bat
For Linux execute run.sh
